Arbitrary Precision Integers with BigInt
Es una nueva primitiva de JavaScript que puede representar numeros enteros con precision arbitraria a diferencia de la primitiva Numero que solo puede almacenar de forma segura valores enteros hasta 2 ^ 53.
Sintaxis
Numero(Normal) = 156 BigInt = 156n
Conversion
const aBigInt = 11n;
const aNumber = 156;
const aBigInt = BigInt(aNumber);
aBigInt === 156n // true
parseFloat()y parseInt()también puede convertir un BigInt en un Número.

Es una secuencia de caracteres del conjunto de caracteres de origen entre comillas dobles. Se utilizan para representar una secuencia de caracteres que forman una cadena terminada en nulo.

No se representa a sí misma cuando se usa dentro de un carácter o literal de cadena , pero se traduce a otro carácter o una secuencia de caracteres que puede ser difícil o imposible de representar directamente.

Symbols
Es un tipo de datos cuyos valores son únicos e immutables. Dichos valores pueden ser utilizados como identificadores (claves) de las propiedades de los objetos. Cada valor del tipo Symbol tiene asociado un valor del tipo String o Undefined que sirve únicamente como descripción del símbolo.

DIA 3

Primary Expressions:
Las expresiones primarias en JavaScript son valores constantes o literales , ciertas palabras clave de lenguaje y referencias a variables.
Object and Array Initializers:
Los inicializadores de objetos y matrices son expresiones cuyo valor es un objeto o matriz recién creado. Estas expresiones de inicializador a veces se denominan "literales de objeto" y "literales de matriz".
Array Initializers:
Un inicializador de matriz es una lista de expresiones separadas por comas contenidas entre corchetes. El valor de un inicializador de matriz es una matriz recién creada. Los elementos de esta nueva matriz se inicializan con los valores de las expresiones separadas por comas
Object Initializers:
son como expresiones de inicializador de matriz , pero los corchetes se reemplazan por corchetes, y cada subexpresión tiene como prefijo un nombre de propiedad y dos puntos
